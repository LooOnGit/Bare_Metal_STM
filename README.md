# Bare_Metal_STM

## Clone project
```bash
$ git clone https://github.com/LooOnGit/Bare_Metal_STM.git
$ cd Bare_Metal_STM
```

## Run document
### General file 
```bash
$ doxygen Doxyfile
```
### Show document
```bash
$ start Docs\html\index.html
```
## Run Application
### Build App
```bash
$ make
```
### Clean build
```bash
$ make Clean
```
### Flash to board
```bash
$ make Flash
```
